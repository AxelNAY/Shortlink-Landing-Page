What is a Shortlink ?

A shortlink (or URL shortener) is a tool that converts a long URL into a shorter, more manageable link while still directing to the same webpage.

Creation of a Shortlink

The user can enter a link in the text box and select the lifespan of the shortlink, ranging from 1 to 120 days. When they press "Generer", the shortlink is created and immediately available for use.
